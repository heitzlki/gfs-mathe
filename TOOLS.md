### LaTex

#### Texmaker

```
sudo apt-get install texmaker
```

#### fix beamer.cls issue

```
sudo apt install texlive-latex-recommended
```

#### Babel german

> Paket suchen

```
sudo apt-cache search texlive german
```

> Paket installieren

```
sudo apt-get install texlive-lang-german
```
